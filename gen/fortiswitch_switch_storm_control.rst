fortiswitch_switch_storm_control -- Configure excess switch traffic (storm control) in Fortinet's FortiSwitch
+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

.. versionadded:: 1.0.0

.. contents::
   :local:
   :depth: 1


Synopsis
--------
- This module is able to configure a FortiSwitch device by allowing the user to set and modify switch feature and storm_control category. Examples include all parameters and values need to be adjusted to datasources before usage. Tested with FOS v7.0.0



Requirements
------------
The below requirements are needed on the host that executes this module.

- ansible>=2.15

Examples
--------

.. code-block:: yaml+jinja
    
    - name: Configure excess switch traffic (storm control).
      fortinet.fortiswitch.fortiswitch_switch_storm_control:
          switch_storm_control:
              broadcast: "enable"
              burst_size_level: "4"
              rate: "5"
              unknown_multicast: "enable"
              unknown_unicast: "enable"
