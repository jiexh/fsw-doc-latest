fortiswitch_log_memory_global_setting -- Global settings for memory log in Fortinet's FortiSwitch
+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

.. versionadded:: 1.0.0

.. contents::
   :local:
   :depth: 1


Synopsis
--------
- This module is able to configure a FortiSwitch device by allowing the user to set and modify log_memory feature and global_setting category. Examples include all parameters and values need to be adjusted to datasources before usage. Tested with FOS v7.0.0



Requirements
------------
The below requirements are needed on the host that executes this module.

- ansible>=2.15

Parameters
----------


.. raw:: html

    <ul>
    <li> <span class="li-head">enable_log</span> - Enable/Disable logging for task. <span class="li-normal">type: bool</span> <span class="li-required">required: false</span> <span class="li-normal">default: False</span> </li>
    <li> <span class="li-head">member_path</span> - Member attribute path to operate on. <span class="li-normal">type: str</span> </li>
    <li> <span class="li-head">member_state</span> - Add or delete a member under specified attribute path. <span class="li-normal">type: str</span> <span class="li-normal">choices: present, absent</span> </li>
    <li> <span class="li-head">log_memory_global_setting</span> - Global settings for memory log. <span class="li-normal">type: dict</span> </li>
        <ul class="ul-self">
        <li> <span class="li-head">full_final_warning_threshold</span> - Log full final warning threshold(3-100), the default is 95. <span class="li-normal">type: int</span> </li>
        <li> <span class="li-head">full_first_warning_threshold</span> - Log full first warning threshold(1-98), the default is 75. <span class="li-normal">type: int</span> </li>
        <li> <span class="li-head">full_second_warning_threshold</span> - Log full second warning threshold(2-99), the default is 90. <span class="li-normal">type: int</span> </li>
        <li> <span class="li-head">hourly_upload</span> - Whether to enable scheduled upload hourly. <span class="li-normal">type: str</span> <span class="li-normal">choices: enable, disable</span> </li>
        <li> <span class="li-head">max_size</span> - Maximum size(byte) in memory buffer log. <span class="li-normal">type: int</span> </li>
        </ul>
    </ul>


Examples
--------

.. code-block:: yaml+jinja
    
    - name: Global settings for memory log.
      fortinet.fortiswitch.fortiswitch_log_memory_global_setting:
          log_memory_global_setting:
              full_final_warning_threshold: "50"
              full_first_warning_threshold: "49"
              full_second_warning_threshold: "49"
              hourly_upload: "enable"
              max_size: "7"
