fortiswitch_system_bug_report -- Configure bug report in Fortinet's FortiSwitch
+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

.. versionadded:: 1.0.0

.. contents::
   :local:
   :depth: 1


Synopsis
--------
- This module is able to configure a FortiSwitch device by allowing the user to set and modify system feature and bug_report category. Examples include all parameters and values need to be adjusted to datasources before usage. Tested with FOS v7.0.0



Requirements
------------
The below requirements are needed on the host that executes this module.

- ansible>=2.15

Examples
--------

.. code-block:: yaml+jinja
    
    - name: Configure bug report.
      fortinet.fortiswitch.fortiswitch_system_bug_report:
          system_bug_report:
              auth: "yes"
              mailto: "<your_own_value>"
              password: "<your_own_value>"
              server: "192.168.100.40"
              username: "<your_own_value>"
              username_smtp: "<your_own_value>"
