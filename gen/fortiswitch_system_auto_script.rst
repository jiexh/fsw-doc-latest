fortiswitch_system_auto_script -- Configure auto script in Fortinet's FortiSwitch
+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

.. versionadded:: 1.0.0

.. contents::
   :local:
   :depth: 1


Synopsis
--------
- This module is able to configure a FortiSwitch device by allowing the user to set and modify system feature and auto_script category. Examples include all parameters and values need to be adjusted to datasources before usage. Tested with FOS v7.0.0



Requirements
------------
The below requirements are needed on the host that executes this module.

- ansible>=2.15

Examples
--------

.. code-block:: yaml+jinja
    
    - name: Configure auto script.
      fortinet.fortiswitch.fortiswitch_system_auto_script:
          state: "present"
          system_auto_script:
              interval: "15778800"
              name: "default_name_4"
              output_size: "512"
              repeat: "32767"
              script: "<your_own_value>"
              start: "manual"
